## Installation

### Plugin managers

```viml
" vim-plug
Plug 'PlagaMedicum/ostis-syntax.vim'

" Vundle
Plugin 'PlagaMedicum/ostis-syntax.vim'

" NeoBundle
NeoBundle 'PlagaMedicum/ostis-syntax.vim'
```

### Manual

Run [./install.sh](install.sh) to copy files in `~/.vim`.
